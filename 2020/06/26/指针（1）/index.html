指针（1）

发表于2020-06-26|更新于2020-06-27|学习笔记C语言
|阅读量:

指针是一种数据类型，p是指针类型变量，用来指向一个变量的地址

sizeof()指针类型在内存中的大小，在32位操作系统中所有指针大小都是4个字节，在64位都是8个字节

野指针

野指针就是指针指向的位置是不可知的（随机的、不正确的、没有明确限制的）指针变量在定义时如果未初始化，其值是随机的，指针变量的值是别的变量的地址，意味着指针指向了一个地址是不确定的变量，此时去解引用就是去访问了一个不确定的地址，所以结果是不可知的。

空指针

在变量声明的时候，如果没有确切的地址可以赋值，为指针变量赋一个 NULL 值是一个良好的编程习惯。赋为 NULL 值的指针被称为空指针。NULL 指针是一个定义在标准库中的值为零的常量。一般用作于条件判断。

零指针（万能指针）0-255都是系统保留

零值指针，是值为0的指针，可以是任何一种指针类型，可以是通用变体类型void*，也可以是char*，int*等等，可以指向任何类型。

const修饰指针

如果const直接修饰变量，指针可以修改。如果const修饰指针类型（const int * p)，不能改变指针变量指向的内存地址的值，但是可以改变指向的内存地址。如果const修饰指针变量（int * const p）能修改指针变量指向地址的值，但不能修改指向的内存地址。==同时修饰（const int const *p）==两个都不能修改。

指针和数组、指针运算

  1. 数组名是数组的首地址，这是一个常量
  2. 指向数组的指针 
      格式int arr [10] = { 0 } ;  
          int * p = arr;
      当操作指针的时候 间接操作了数组 arr[i] = p[i];
  3. 指针的降级操作 ,取当前地址的值：p[5]、*p 
  4. 对指向数组的指针进行加减操作，可能会导致数组下标越界 。
  5. 相同的类型的指针相减  结果是两个指针相差的长度
    区别：数组名通过sizeof可以求出数组大小，指针只包含数组的首地址信息

指针数组

  1. 存储char *类型的地址数组
  2. 指针数组，它是数组，数组的每个元素都是指针类型，指向地址。

多级指针

  1. 如果二级指针前面加一个* 代表一级指针的值
  2. 二级指针前面加** 代表指针指向一级指针指向地址的值，加*降维度
  3. 如果n级指针在前面加n个*就是指针指向一级指针指向地址的值
